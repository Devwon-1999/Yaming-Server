## 🌟 Overview
**Yaming-Server** is a **Node.js + MySQL** backend that powers the Yaming AI-based meal recommendation mobile app.  
It provides REST APIs for food recognition results, meal recommendations, and user management.

---

## 🧩 Features
- 🌐 **RESTful API** – Handles requests from the Kotlin Android client.  
- 🧠 **AI Integration** – Connects with AI inference services for food recognition.  
- 🍽️ **Recommendation Logic** – Generates personalized meal suggestions based on nutritional data.  
- 💾 **MySQL Database** – Manages user info, food data, and recommendation history.  
- 🚀 **Deployed via Vercel / Node Environment** – Simple, lightweight deployment setup.
